# 🚗 ETEC - Site FakeNews

Projeto desenvolvido em sala de aula com o objetivo de criar um site sobre Fake News.

## 📖 Introdução

Este projeto tem como foco explorar o conteúdo sobre fake new

## 🎯 Objetivo

Criar um site informativo e visualmente atrativo que mostre coisas sobre fake news, utilizando os conhecimentos adquiridos em sala de aula sobre HTML, CSS e posicionamento de elementos.

## ✅ Requisitos do Projeto

- [x] Apresentar **no mínimo 5 carros antigos**
- [x] Apresentar **Elementos cards.**
- [x] Incluir uma **introdução** sobre o tema
- [x] Utilizar **CSS de forma criativa**
- [x] Aplicar **posicionamento de elementos**
- [x] Demonstrar **criatividade** no design
- [x] Incluir os **nomes dos desenvolvedores** do site
